# simple-birthday-webapp
Simple webapp to track birthdays
---

## DOCKER IMAGE
Pull & Run commands:<br>
```bash
sudo docker pull matiasrgt/flask-birthday
```
```bash
sudo docker run -p 5000:5000 matiasrgt/flask-birthday
```
